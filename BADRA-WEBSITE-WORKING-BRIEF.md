# Badra Website Working Brief

This file is the running memory for the website project.
It should be updated as decisions are made.
Each new round can append a dated section rather than rewriting the whole document.

## 2026-03-26

### Project Goal
Build a world-class luxury website for Badra that functions as:

- a booking platform
- a portfolio
- an authority platform
- a media / press archive
- a business ecosystem

The website is not only for dance.
It must present Badra as a global artist whose work spans multiple disciplines.

### Core Identity
Working core statement:

> Badra is a global artist, actress, and performer.

Expanded truth:

- one of the biggest stars in belly dance
- more than a belly dancer
- artist
- actress
- teacher
- businesswoman
- co-directs / shapes vision, content, events, and creative work

### Primary Website Objective
The main purpose of the website is:

- to help people understand who Badra is
- to make her easier to book
- to make all her work legible in one place

Primary audience:

- people who need to book her
- business-first audience
- clients, organizers, collaborators, and people looking for her services

### Booking Categories Currently Locked
These are the main booking categories discussed so far:

1. Live performance dancer
2. Actress
3. Event creator
4. Creative director
5. Teacher

Homepage emphasis order currently discussed:

1. Live performance dancer
2. Actress
3. Teacher

### Emotional / Visual Direction
First-screen feeling should be:

- sensual
- untouchable
- cinematic

Other deeper truths:

- visual presence is a major part of her icon status
- international reach matters
- versatility matters
- her brain, discipline, and persistence are central to the truth of who she is

### Hero Direction Decisions
Current decisions for the first screen:

- face-led, not silhouette-led
- face should be partially hidden by veil or fabric
- mystery should remain
- it should still be clearly Badra
- still image, not video background
- performance portrait
- intense and cinematic
- black and red direction
- Egyptian sunset light feeling
- centered hero text
- booking categories displayed in one horizontal line under the hero statement
- main hero CTA should be a luxury button
- CTA should feel visible and intentional, but not annoying
- CTA color direction: red and powerful
- hero should use one CTA only
- hero should include a subtitle above the main statement
- subtitle direction: cinematic-led
- current preferred subtitle idea: performance icon
- after the hero there should be a visual pause before biography begins
- the visual pause should be a large image
- the visual pause image can be either a close detail or an atmospheric scene
- hero should rely on one unforgettable image
- hero image should feel mysterious and textured
- her face should still be clearly visible inside that mystery
- the homepage background outside the hero should stay very dark and quiet

Hero copy direction:

- first headline should say who she is
- not abstract poetry first
- not sales-first language first

Current working identity line:

> Badra is a global artist, actress, and performer.

What comes directly under the hero statement:

- booking categories

Main hero CTA:

- View Portfolio

Reason:

- the visitor needs to understand immediately why they are on the website
- booking intent is important, but portfolio is the most useful first action until a stronger showreel exists

### Biography Section Direction
The first biography block should communicate:

- everything she does

The four most important roles for that first biography framing:

1. Dancer
2. Actress
3. Teacher
4. Businesswoman

Dominant homepage force among those:

- dancer

But the overall framing should not be:

- performer only
- businesswoman only

It should feel:

- icon first

Biography copy tone:

- a mix of editorial and business-clear

Biography structure preference:

- one strong paragraph
- portrait + text
- portrait should feel formal and iconic
- portrait crop direction: three-quarter portrait
- biography section title: Presence, Work, and Vision
- biography layout: portrait on the left, text on the right
- biography section should end with a link to the About page
- that link should feel like a small elegant button

### Header / Navigation Direction
Header should feel:

- luxury
- almost invisible
- smooth
- quiet
- always present
- helpful without dominating

Header should support layered navigation:

- top-level items
- submenus inside sections
- information organized under bigger worlds

Current direction:

- Logo = Home
- About
- Portfolio
- Services
- Academy
- Shop
- Press
- Book CTA

Important decisions:

- Academy and Shop should be separate top-level sections
- not nested under Services
- Book should be a separate CTA in the header
- Book is the most important action and can have its own identity
- header CTA should be longer and more intentional than just "Book"
- it should feel like a real high-value contact / inquiry action
- recommended header CTA wording: Work With Badra
- on desktop, navigation submenus should open on hover
- dropdown style direction: full-width editorial drawers
- mobile navigation should feel app-like
- mobile menu should open as an immersive mode, not a small panel
- mobile menu background should use a dark black / deep red abstract animation
- white text over high-opacity dark layers
- subtle texture and soft cinematic atmosphere behind the menu
- mobile menu links should feel clean and practical
- Shop is a real priority, but not a homepage focus
- Shop should exist as a top-level section / link
- users can be redirected directly to the shop
- for now, the website should not push Shop as one of the main homepage conversion goals

## 2026-03-27

### Design System Refinement Implemented
The live website design was upgraded while keeping the existing structure.

Refinements now locked:

- typography direction tightened to editorial serif + clean modern sans rather than browser-default luxury styling
- dark surfaces now rely more on tonal layering and light-leak gradients, less on obvious boxed borders
- gold is used as the authority accent for headings, highlights, and framing
- red stays reserved for high-intent actions and pressure points such as booking CTAs
- imagery should feel framed, cinematic, and ceremonial rather than dropped into generic cards
- navigation should read quieter, more invisible, and more premium while remaining useful
- section spacing is intentionally larger to preserve a luxury rhythm and avoid crowding
- hover and interaction states should feel subtle, heavy, and controlled rather than playful
- should not look cheap
- should not look like a typical belly dance website

- overall experience does not yet feel imagined enough
- current design does not yet feel like the best website in the world for a major artist

### Brand Proof Direction
Current decision for the post-biography proof section:

- brands awareness should come early
- it is stronger than portfolio or press as an immediate business trust signal

How it should appear:

- logos
- maybe short names
- each can link to the partner / brand page

Current preferred direction:

- logos + names + outbound links
- white / neutral logos on a dark background
- clean horizontal row with subtle motion
- brands section should feel almost silent

### Services Section Direction
After the biography block and brand proof, the next homepage section should be:

- services overview

Reason:

- this is the section that answers the booking client's practical question
- the visitor needs to understand what Badra can be hired for
- portfolio belongs more naturally inside the dedicated Portfolio area

Services text / layout preference:

- very structured and clear
- no doubt
- artistic direction lives in the visual treatment, not in vague wording
- homepage services should be shown as five elegant cards
- each service card should use a title + one-line explanation
- services section should have a strong title
- services section title: Ways to Work With Badra
- each homepage service card should link to its own service detail page
- do not send users straight to Booking from the homepage service cards
- homepage service cards should include pictures
- service detail pages should be fully independent based on the subject
- the dancer page should feel the strongest and most luxurious
- the event creator page should feel the most different from the dancer page
- event creator page mood: immersive experience design
- teacher page mood: warm mentorship
- actress page mood: cinematic, elegant, and prestigious
- creative director page mood: sharp and visionary

### Homepage Overall Framing
The homepage should feel:

- icon first
- ecosystem second

Reason:

- the first impression should establish Badra as a major figure
- the ecosystem of services, roles, and opportunities should unfold after that

### Proof Sequence Direction
After the services overview, the homepage should move into proof of authority.

Current preferred order:

1. Press / media
2. International reach

Reason:

- press and media read faster as external validation
- international reach strengthens that proof after credibility is already established

### Homepage Press / Media Direction
The homepage press section should be:

- a teaser, not the full archive
- a small editorial block
- one quote or highlighted mention
- a clear link to the full Press page
- a visible path for press contact / article inquiries
- the section title should be visible
- the section title tone should feel editorial

Reason:

- the homepage should signal media authority without becoming overloaded
- the full detail should live inside the dedicated Press page

### International Reach Direction
The homepage international section should communicate global reach through:

- cities first
- countries as supporting context

Preferred presentation direction:

- a visual map
- editorial in tone
- clean integrated map
- minimal and refined

Reason:

- it makes the international scale feel immediate
- cities feel more concrete and bookable than countries alone

### Acting Placement Direction
The acting part should not appear later as a separate homepage block.

It should live inside:

- biography / identity framing

Reason:

- acting is part of who she is
- not a detached later feature

### Calendar Direction
After international reach, the homepage should move to:

- calendar / upcoming travel

Homepage calendar treatment:

- a simple teaser
- links to the full calendar page
- homepage calendar teaser tone: editorial and elegant
- full calendar page tone: clear and practical
- homepage calendar teaser title: Where She Moves Next

Reason:

- useful for booking intent
- keeps the homepage light while still showing movement and availability

### Homepage Closing Direction
The final homepage section before the footer should be:

- a Shop teaser
- with a short closing statement
- shop teaser should feel like a quiet link to e-commerce

Reason:

- Shop is real and important
- but it should not dominate the homepage earlier in the journey
- it can function well as a final secondary invitation before the footer

Closing statement tone:

- sensual and mysterious

### Service Card Image Direction
Homepage service cards should use:

- pictures, not typography only
- stylized and atmospheric images
- imagery that supports the service mood rather than literal explanation
